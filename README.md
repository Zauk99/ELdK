Diario de Kanto - Frontend (Portal de Noticias y Pokédex)

Este repositorio contiene la interfaz de usuario y la lógica de cliente para "El Diario de Kanto", una plataforma web Full-Stack orientada a los fans de Pokémon. La aplicación incluye un portal de noticias, una Pokédex interactiva, minijuegos clásicos y gestión de perfiles de usuario.

⚠️ AVISO IMPORTANTE: Este frontend consume una API REST propia desarrollada desde cero. Puedes consultar el código fuente del backend, la base de datos y los endpoints en el siguiente repositorio: (https://github.com/Zauk99/ELdK_Api)

Stack Técnico:

Backend for Frontend (BFF): Java, Spring Boot

Motor de Plantillas: Thymeleaf

Frontend: HTML5, CSS3, Vanilla JavaScript

Gestión de dependencias: Maven

Capturas de la Aplicación:
[Una de la portada, otra de la Pokédex y otra de algún minijuego como el Voltorb Flip]

Características Principales:

Consumo dinámico de la API REST mediante Spring Boot y renderizado con Thymeleaf.

Sistema de usuarios (Login, registro y perfiles personalizados).

Pokédex interactiva con información detallada de los Pokémon.

Sección de Minijuegos funcionales (Adivina el Pokémon, Voltorb Flip).

Diseño web responsive adaptado a múltiples dispositivos.

Cómo ejecutar el proyecto en local:

Clona este repositorio en tu equipo.

Asegúrate de tener la API REST (Backend) en ejecución en tu máquina local.

Importa el proyecto en tu IDE favorito (IntelliJ IDEA, Eclipse, VS Code).

Ejecuta la clase principal WebApplication.java.

Accede a la aplicación desde tu navegador en localhost (normalmente puerto 8080 o el que tengas configurado).
